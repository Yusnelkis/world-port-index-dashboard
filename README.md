# World Port Index Dashboard

Interactive dashboard exploring 3,824 ports across 196 countries with maps, charts, and detailed port intelligence.

## Live Demo
https://yusnelkis.github.io/world-port-index-dashboard/

## Features
- Interactive Leaflet map with 3,824 port markers
- - Chart.js visualizations (bar, doughnut charts)
  - - Filterable by country, harbor size, type
    - - Sortable data table
      - - Responsive design
